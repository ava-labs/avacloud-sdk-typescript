# Rpc
(*rpc*)

## Overview

Interact with chain nodes using JSON-RPC.

### Available Operations

* [rpc](#rpc) - Calls JSON-RPC method

## rpc

Calls JSON-RPC method.

### Example Usage

```typescript
import { GlacierSDK } from "@avalabs/glacier-sdk";

const glacierSDK = new GlacierSDK();

async function run() {
  const result = await glacierSDK.rpc.rpc({
    chainId: "43114",
  requestBody:     [
        {
          method: "<value>",
        },
      ],
  });

  // Handle the result
  console.log(result)
}

run();
```
